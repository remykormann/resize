# Resize (PMMP5 Plugin)
<img width="2559" height="1494" alt="image" src="https://github.com/user-attachments/assets/ccf5c8d8-4ea5-4600-ad34-d87b8cc43127" />
Resize is a plugin for PMMP5 that allows you to change the player's size.

Unlike other existing plugins, which only modify the player size without adjusting the first-person view, Resize aims to provide a complete system. This means both the player model and the camera perspective are properly handled for a more consistent experience.
